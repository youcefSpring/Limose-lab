## Step 3: Create Migrations

- **Goal:** Generate migration files for the database.
- **Reference File:** 04-DatabaseSchema.sql
- **Tasks:**
  1. Analyze the schema definitions in 04-DatabaseSchema.sql.
  2. Use `php artisan make:migration` to create corresponding migration files.
  3. Place all migration files in the `database/migrations` directory.

- **Command/Location:** 
  - Use `make:migration` command.
  - Save in `database/migrations`.
